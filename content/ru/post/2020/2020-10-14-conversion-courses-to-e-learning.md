## Переход на электронные учебные курсы {#переход-на-электронные-учебные-курсы}

-   Текущие учебные курсы необходимо перевести в формат электронного учебного курса.
-   Курсы предлагается располагать на платформе Moodle.
-   Необходимо сформировать структуру электронного учебного курса.
-   Из требований к электронному учебному курсу необходимо сформировать требования к настройке Moodle.

#### Материалы для изучения в текстовом формате {#материалы-для-изучения-в-текстовом-формате}

<!--list-separator-->

-  Состав:

    <!--list-separator-->

    -  Текстовый учебный материал раздела:

        <!--list-separator-->

        -  авторский текста;

        <!--list-separator-->

        -  фрагмент учебника (с учётом лицензионных ограничений);

        <!--list-separator-->

        -  отражает основные положения изучаемого тематического раздела.

    <!--list-separator-->

    -  Формат: `.pdf`, `.djvu`, `.html`.

    <!--list-separator-->

    -  Ресурсы Moodle:

        <!--list-separator-->

        -  «Книга»,

        <!--list-separator-->

        -  «Лекция»,

        <!--list-separator-->

        -  «Гиперссылка»,

        <!--list-separator-->

        -  «Файл» (`pdf`, `djvu`),

        <!--list-separator-->

        -  «Папка».


#### Материалы для изучения в мультимедиа формате: {#материалы-для-изучения-в-мультимедиа-формате}

<!--list-separator-->

-  Состав:

    <!--list-separator-->

    -  Видео:

        <!--list-separator-->

        -  авторская запись в лекционной аудитории,

        <!--list-separator-->

        -  скринкаст,

        <!--list-separator-->

        -  самозапись,

        <!--list-separator-->

        -  студийная запись,

        <!--list-separator-->

        -  запись вебинара.

    <!--list-separator-->

    -  Интерактивные материалы.

    <!--list-separator-->

    -  Озвученные презентации или видеозапись озвученной презентации.

    <!--list-separator-->

    -  Аудио материалы («подкасты»).

<!--list-separator-->

-  Должны быть обеспечены качественные звук и изображение.

<!--list-separator-->

-  Видео размещается в виде отдельных тематических видео-блоков, длительностью до 60 мин. (лучше 5-15 мин.).

<!--list-separator-->

-  Формат: ссылка на Youtube, `.mp4` (видео), `.mp3` (аудио).

<!--list-separator-->

-  Ресурсы Moodle:

    <!--list-separator-->

    -  «Файл» (`.mp4`, `.mp3`),

    <!--list-separator-->

    -  «Гиперссылка» (внешние ресурсы).


#### Комплект материалов для лабораторных работ (если лабораторные работы предусмотрены учебным планом) {#комплект-материалов-для-лабораторных-работ--если-лабораторные-работы-предусмотрены-учебным-планом}

<!--list-separator-->

-  Состав:

    <!--list-separator-->

    -  перечень тем лабораторных работ;

    <!--list-separator-->

    -  задания для лабораторных работ;

    <!--list-separator-->

    -  видеозапись описания выполнения лабораторной работы;

    <!--list-separator-->

    -  список рекомендуемых источников для подготовки к лабораторным работам;

    <!--list-separator-->

    -  описание виртуального лабораторного комплекса;

    <!--list-separator-->

    -  инструкции по выполнению и оформлению лабораторных работ;

    <!--list-separator-->

    -  задания для допуска к лабораторной работе;

    <!--list-separator-->

    -  иные материалы по усмотрению преподавателя.

<!--list-separator-->

-  Формат: `.pdf`, `.html`, `.doc`, `.xls`, `.ppt` и т.д..

<!--list-separator-->

-  Ресурсы Moodle:

    <!--list-separator-->

    -  «Файл» (`pdf`, `doc`, `xls`, `ppt` и др.),

    <!--list-separator-->

    -  «Страница» (`html`),

    <!--list-separator-->

    -  «Гиперссылка» (для размещения ссылок на внешние ресурсы).


#### Комплект материалов для аудиторного занятия (практика/семинар) {#комплект-материалов-для-аудиторного-занятия--практика-семинар}

    <!--list-separator-->

#### Комплект материалов для индивидуальной работы студентов {#комплект-материалов-для-индивидуальной-работы-студентов}

    <!--list-separator-->

    -  методические указания, инструкции и требования к оформлению и выполнению заданий;

    <!--list-separator-->

    -  критерии оценивания;

    <!--list-separator-->

    -  иные материалы по усмотрению преподавателя.

<!--list-separator-->

-  Формат: `.pdf`, `.html`, `.doc`, `.xls`, `.ppt` и т.д.

<!--list-separator-->

-  Ресурсы Moodle:

    <!--list-separator-->

    -  «Задание».


#### Комплект заданий для совместной работы студентов {#комплект-заданий-для-совместной-работы-студентов}

<!--list-separator-->

-  Состав:

    <!--list-separator-->

    -  теоретические и практические задания;

    <!--list-separator-->

    -  методические указания, инструкции и требования к оформлению и выполнению заданий;

    <!--list-separator-->

    -  инструкции для взаимного рецензирования/взаимного оценивания;

    <!--list-separator-->

    -  критерии оценивания;

    <!--list-separator-->

    -  иные материалы по усмотрению преподавателя.

<!--list-separator-->

-  Формат: `.pdf`, `.html`, `.doc`, `.xls`, `.ppt` и т.д.

<!--list-separator-->

-  Ресурсы Moodle:

    <!--list-separator-->

    -  «Форум»,

    <!--list-separator-->

    -  «Wiki»,

    <!--list-separator-->

    -  «Семинар»,

    <!--list-separator-->

    -  «Опрос».


#### Материалы для контроля результатов обучения {#материалы-для-контроля-результатов-обучения}

<!--list-separator-->

-  Состав:

    <!--list-separator-->

    -  вопросы для самопроверки;

    <!--list-separator-->

    -  тесты для текущего контроля:

        <!--list-separator-->

        -  не менее 10 тестовых вопросов для каждого тематического раздела

        <!--list-separator-->

        -  рекомендуемое соотношение типов заданий:

            <!--list-separator-->

            -  на выбор единственного ответа - 40%,

            <!--list-separator-->

            -  на выбор множественного ответа - 20%,

            <!--list-separator-->

            -  на установление соответствия - 10%,

            <!--list-separator-->

            -  на установление правильной последовательности - 20%,

            <!--list-separator-->

            -  вопросов открытого типа - 10%.

<!--list-separator-->

-  Ресурсы Moodle:

    <!--list-separator-->

    -  «Тест».

<!--list-separator-->

-  В случае, если тестирование реализовано средствами встроенной в Moodle подсистемы тестирования, тесты создаются путём импорта из банка вопросов электронного курса.

<!--list-separator-->

-  Тестовые задания могут быть включены в состав интерактивных учебных пособий.
